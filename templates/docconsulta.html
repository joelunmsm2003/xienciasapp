Respuestas de las llamadas usando la API de Xiencias!

En unos pocos pasos, nosotros te explicaremos como consultar las llamadas realizadas usando la API de Xiencias

Primeramente, tu necesitaras una cuenta en Xiencias. Cuando tu tengas una cuenta tu obtendras un usuario y password

Luego tu usuario y password sera codificado en base64 como sigue:

  • Combina el usuario y password en un string usuario:password

  • Codifica el resultado usando Base64


    • Ejemplo

    Username: root

    Password: 123

    Base64 codificacion string: cm9vdDoxMjM=

    Consulta total de llamadas

    http://xiencias.com:9000/consultablaster/

    Parametros

    Variables

    Significado

    auth

    String

    Codifacion Base64 del usuario y password

    Consulta de llamadas por lote

    http://xiencias.com:9000/consultaloteblaster/

    Parametros

    Variables

    Significado

    auth

    String

    Codifacion Base64 del usuario y password

    lote

    String

    Nombre del lote
