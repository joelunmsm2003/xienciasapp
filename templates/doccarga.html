Envio Voip

Envio de numeros telefonicos usando la API de Xiencias!

En unos pocos pasos, nosotros te explicaremos como enviar llamadas masivas usando la API de Xiencias

Primeramente, tu necesitaras una cuenta en Xiencias. Cuando tu tengas una cuenta tu obtendras un usuario y password

Luego tu usuario y password sera codificado en base64 como sigue:

  • Combina el usuario y password en un string de la siguiente forma usuario:password

  • Codifica el resultado usando Base64


    • Ejemplo

    Usuario: root

    Password: 123

    Base64 codificacion string: cm9vdDoxMjM=

    Definicion

    https://xiencias.com:9000/blaster/

    Parametros

    Variables

    Significado

    auth

    String

    Codifacion Base64 del usuario y password

    base

    Archivo

    Es el archivo excell a subir donde en la primera columna estan los numeros que recibiran las llamadas

    La direccion de destino debe tener el formato (Ejemplo: 980729169).

    audio1

    Archivo

    Representa el archivo de audio en mp3 que sera enviado en las llamadas

    nombre

    String

    Representa el nombre del lote a enviar
