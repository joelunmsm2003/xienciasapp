Introducción

Bienvenido a la documentación de la API de Voz!

Este documento le proveera instrucciones sobre cómo integrar rápidamente los servicios de llamada de voz en varias soluciones

mediante el uso de la interfaz de programación de aplicaciones . La API HTTP se puede utilizar para el envío de llamadas de voz VOIP, reportes de respuesta, etc

La API de Xiencias se basa en estándares REST, que le permite utilizar el navegador para acceder a URLs.

Con el fin de interactuar con nuestra API, cualquier cliente HTTP en cualquier lenguaje de programación podra utilizarlo.


Base URL

Presentar todas las peticiones a la URL base. Todas las solicitudes son enviadas con metodo HTTP POST, por lo que el tráfico se cifra y la privacidad está garantizada.

Base URL: https://xiencias.com:9000

Autorizacion

Nosotros utilizamos la autorizacion usando el username y password codificado en Base64 aqui

La autorizacion es construida como sigue:

usuario y password son combinados en el siguiente string username:password

El resultado del string es codificado en Base64


Content-Type

API Voip soporta JSON y acepta criterios que deben ser especificados. Si no se especifica el tipo de contenido usted recibirá un error general.

Acepta el encabezado: application/json
